# World Cafe - Personal Records One-Sheet

A beautifully designed promotional one-sheet for NPR's World Cafe radio show "Personal Records" conversation series.

## Project Structure

- `index.html` - Main HTML file with semantic markup
- `styles.css` - All CSS styling separated from HTML
- Modern, responsive design with print-optimized layout

## Features

- **Responsive Design**: Works on desktop, mobile, and print
- **Custom Typography**: Uses Figtree font for modern, readable text
- **Color-Coded Elements**: Artist names, hosts, and content sections use World Cafe brand colors
- **Print Optimized**: Special print styles for physical one-sheets
- **Accessible**: Semantic HTML with proper heading structure

## Color Palette

- Cream Background: `#f6f0e3`
- Text: `#2e2a24`
- Accent Teal: `#629495`
- Accent Gold: `#e39229`
- Accent Rust: `#c24732`

## Usage

Simply open `index.html` in any modern web browser. The page is designed to be self-contained and works without any build process.

## Print

The page includes special print styles optimized for 8.5" x 11" paper. Use your browser's print function to create physical copies.
